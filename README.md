# 🎨 Mixora – Color ASMR Lab

Mixora is a modern, interactive, ASMR-inspired color mixing web application where users can blend colors gradually with satisfying animations and sound effects.

---

## 🌟 Features

### 🎨 Interactive Color Mixing
- Select up to 3 colors
- Drop them into a mixing area
- Gradually mix colors with each click/tap
- Realistic blending (NOT instant)

---

### 🔁 Progressive Mixing System
- Each click increases mixing progress
- Smooth transition from separate colors → fully blended
- Visual swirl and liquid effects

---

### 💧 Liquid Animation
- Fluid, paint-like behavior
- Ripple and swirl effects
- Smooth transitions

---

### 🔊 ASMR Sound Effects
- Water drop / paint mixing sounds
- Sound on each click
- Optional sound toggle

---

### 🎛️ Controls
- Mix (Click/Tap)
- Reset
- Random Color Generator
- Save Favorite Colors

---

### ❤️ History & Favorites
- View recent mixes
- Save favorite colors
- HEX color display

---

### 📱 Responsive Design
- Works on mobile and desktop
- Touch-friendly UI

---

## 🧠 Concept

This project combines:
- ASMR experience  
- Interactive UI  
- Color theory  

Goal: Create a relaxing and addictive visual experience.

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Animation:** Canvas / WebGL / Three.js  
- **Sound:** Web Audio API  

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/shafinahmedbd967-art/mixora-color-asmr.git
````

### 2. Go to project folder

```bash
cd mixora-color-asmr
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the app

```bash
npm run dev
```
